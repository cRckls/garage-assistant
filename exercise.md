###AI Software Developer Take-Home Exercise

###Overview

Build a small application that demonstrates how you use AI as part of your software development workflow.
The application should be relevant to a garage management system and be achievable in a single evening.
Scenario
A garage receives free-text descriptions of vehicle faults from customers. Build a small application that helps transform this information into something more useful for a service adviser or workshop.
The exact functionality, user experience and implementation are for you to decide.
###Expectations
Your submission should include:
- A working application.
- A Git repository containing the complete source code.
- A clear commit history showing the evolution of the solution.
- Documentation explaining how to build and run the application.
- Any supporting material you feel demonstrates your development process.
###Technology
You may use any programming language, framework, AI tooling or language model that you believe is appropriate.
Time
The exercise is intended to take approximately three hours. It is acceptable to leave features unfinished, provided your documentation explains your thinking and any trade-offs you made.
